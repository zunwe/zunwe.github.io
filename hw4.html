HTML
HTML is short for Hyper Text Markup Language and it is a nonproprietary language interpreted by the web browser.
CSS
CSS is short for Cascading Style Sheet and web developers use CSS to style/format their web pages.
FTP
FTP is short for File Transfer Protocol and it's a nectwork protocol that transfer files from computers to another system.
SSH
SSH is short for Secure Shell and it's a network protocol that gives users a secure way to communicate & share data.
List Usage
Unordered List An unordered list is used to present a list of items that not necessarily need to be skown in a specific order. To start this list you use the tag < ul > and < /ul >.
Ordered List An ordered list is used to present a list of items that need to be shown in a specific order. To start this list you use the tag < ol > and < /ol >.
Definition List A definition list is used for the items come in pairs-the term to be defined and the definition(description). This list starts with < dl > and < /dl >.
